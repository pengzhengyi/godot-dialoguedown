# DialogueDown documentation

**DialogueDown** is an engine-agnostic, Markdown-first branching **dialogue
compiler** for C#/.NET. You write dialogue as Markdown; DialogueDown lowers it
through a series of compiler stages into a runtime dialogue graph, keeping the
core free of any game-engine dependency.

> [!NOTE]
> DialogueDown is early-stage. The public API, script language, and runtime model
> may change while the library matures.
