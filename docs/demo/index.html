DialogueDown

An engine-agnostic, Markdown-first dialogue compiler — see a script lowered through its compiler stages as interactive graphs.

This page hosts a live, read-only export of the visualization report for one sample script, examples/gallery.dialogue.md. It runs entirely in your browser — no server.

  • Source — the dialogue script beside a live Markdown preview.
  • Markdown AST — the syntax tree parsed from the source.
  • Dialogue AST — speakers, choices, game calls, and jump indicators.
  • Desugared AST — jumps assembled and default speakers filled in.

Zoom, pan, and collapse the graphs; click a node to inspect the source it came from; and toggle the light/dark theme. Each graph remembers where you left it.
